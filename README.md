## Project Structure
- src/main/java/jdbc/connection/ — Database connection and config classes
- src/main/java/jdbc/exception/ — Custom exception classes
- src/main/java/jdbc/dao/ — CRUD operation classes
- src/main/java/jdbc/model/ — Data model classes
- src/main/java/jdbc/ui/ — Swing UI application
- src/test/java/jdbc/ — Test classes
- docs/ — Documentation and guides
- lib/ — JDBC driver and Windows Auth DLL

## How to Run
1. Install Java from https://adoptium.net/
2. Clone this repo
3. Download the JDBC driver and put the jar and dll in lib/
4. Make sure SQL Server Express is running with TCP/IP enabled
5. Compile and run the UI:# -JDBC-Class-Library-Group-1
