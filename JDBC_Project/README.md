│   └── test/java/jdbc/       <- Person 5: Test classes
├── docs/                     <- Person 5: Documentation
├── lib/                      <- External JARs (JDBC drivers, etc.)

## Setup Instructions
1. Clone this repository
2. Add your JDBC driver JAR to the `lib/` folder
3. Update `ConnectionConfig.java` with your database credentials
4. Compile and run
