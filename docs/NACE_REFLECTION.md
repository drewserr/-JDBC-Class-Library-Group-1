# NACE Competencies - EOS Group 1

**Equity & Inclusion:** We split the work evenly so everyone had a fair part to do. We helped each other out when someone got stuck.

**Career & Self-Development:** Working on this project taught us how JDBC actually works and how to connect Java to a database, which is something we would need to know for real jobs.

**Communication:** We used our group chat to keep each other updated on what was done and what still needed work. We also documented our code so other people could follow it.

**Critical Thinking:** We had to figure out how to structure the project so each person's code worked together. We also had to debug issues with the database connection and SQL queries.

**Professionalism:** We used GitHub to keep our code organized and made sure everything was submitted on time with proper file names and documentation.

**Technology:** We learned how to use Java, JDBC, SQL Server Express, Windows Authentication, Swing for the UI, and Git/GitHub for version control.

**Leadership:** Andrew managed the project, set up the repo, assigned tasks, and made sure everything came together at the end.

**Teamwork:** Each person built their piece and pushed it to GitHub. We relied on each other since Person 3 needed Person 2's code and Person 4 needed Person 3's code.
